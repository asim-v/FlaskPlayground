Rails Considerations

When using Semantic-UI with Ruby on Rails you can take advantage of the helper methods that come with the semantic-ui gem. The icon helper, "semantic_icon('add user')" will generate the appropriate html for the semantic-ui icon. Here is the code for a sign up button w/ an icon for a Rails application that uses devise for authentication...


<%=link_to semantic_icon('add user')+'Sign up', new_user_registration_path, class: 'ui basic teal button' %>

Note the concatination of the semantic icon helper with the string 'Sign up'.

If you don't need a button, you can use the rails helper 'link_to' to create a menu item with an icon like so... <%=link_to semantic_icon('home')+'Home', root_path, class: 'item' %>
